# Nlom Neural Interface

This repository hosts the Nralakk Federation diplomatic mission interface.

## Development

The project uses [Prettier](https://prettier.io/) to maintain a consistent code style.

Format code:

```bash
npx prettier --write "**/*.{html,css,js,md,json}"
```

Check formatting:

```bash
npx prettier --check "**/*.{html,css,js,md,json}"
```
